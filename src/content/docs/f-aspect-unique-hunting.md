### F1 — Purveyor of Curiosities

Gamble 1H weapons or focuses for guaranteed offensive aspects on upgrade. If gamble drops rare not legendary, Cube → Add Affix to upcycle (Pure Primordial Dust, you'll have 800+).

:::caution[Caveat]

Upgrading

ancestral

rare → ancestral legendary can hit the max-roll aspect version. An 850 IP upgrade caps below max. Always start ancestral if you can.
:::

### F2 — Aspect bartering (Pale Marks)

Mercenaries accumulate Pale Marks just from play. Claim from the rapport screen, spend at the Bartering vendor.

| Cache | Contains |
|---|---|
| Mobility / Defensive / Resource Aspect Cache | Legendaries pre-filtered to that category |
| Salvage Materials Cache | 20–30 Forgotten Souls per cache on T12 |
| Boss Summoning Cache | 2–4 lair keys, sometimes greater |
| Masterwork Materials Cache | Masterwork mats |
| Opals & Obols Cache | Opals + Obols |

Open caches → salvage at Blacksmith → game auto-updates the codex if any roll beats your current max. Mercenary leveling unlocks specific cache types (Oriana = opals + masterwork; Subo = boss summons).

### F3 — Recycle uniques

:::tip[Do]
stat lottery

The fastest path to a usable Gohr's, Ramaladni's, or Tibault's: stockpile 3 ancestral copies → Cube → Recycle Uniques → Transmute. New version with re-rolled stats, chance for double GA. Keep the loot filter rule for ancestral build-critical uniques on.
:::

### F4 — Unique power roll

If a unique has good stats but a low aspect %, Cube → Unique Power Roll (Tuned Primordial Dust). Cheap, repeatable. Push to 50%.

### F5 — Plus-skill affixes per slot

Skill trees only roll on specific armor slots.

| Slot | Can roll (Paladin example) |
|---|---|
| Helmet | Valor skills (Aegis, Falling Star, Rally, Shield Charge) |
| Chest | Aura skills |
| Boots | Shouts (Barbarian) / class-specific tree |

**Check yours:** Occultist → enchanter → "View Possible Affixes" → scroll for skills. **Force-roll:** Cube → Add Affix → Skill & Core Stats category (yellow) → Focus Reroll within category until you hit your skill. Cheap, repeatable, massive damage boost most players miss.
