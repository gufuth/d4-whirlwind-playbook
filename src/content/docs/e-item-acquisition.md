:::tip[Big win]
**Lair Bosses > Pit > Helltides > Whispers** for the highest-tier targeted drops. Item power progression is gated by Torment tier unlocks, not by character level after 70.
:::

### E1 — Item power progression

| Stage | Item Power | Where |
|---|---|---|
| Sub-70 | varies | Anywhere |
| 70+ (any difficulty) | 850 baseline | Penitent / overworld |
| T1 entry (Pit 10 clear) | 850 base; ancestral begins to drop | T1+ activities |
| Ancestral gear | 900 | Higher torment + war plan boost |
| Greater affix gear | Random roll on 900 ancestrals | War plans (guaranteed sources), high torment |

:::caution[Caveat]
terminology

Ancestral ≠ Greater Affix. Ancestral = the 900-IP tag. GA = a separate roll on top.
:::

### E2 — Torment tier unlocks

| Tier | Unlock |
|---|---|
| T1 entry | Pit 10 clear |
| T2 | Legendary tempering manuals + Scrolls of Restoration |
| T3 | Set Charms |
| T4+ | Neathiron |
| Higher | Legendary runes, Greater Lair Keys, Primordial Dust |

### E3 — Where things come from

| Item | Best Source |
|---|---|
| Offensive aspects | Purveyor of Curiosities, gamble 1H weapons / focuses (**not gloves**; gloves can roll utility aspects) |
| Defensive / Mobility / Resource aspects | Bartering vendor (Fierra) using Pale Marks, buy the matching aspect cache |
| GA crafting bases | Whisper caches via war plans (left-side material path); 3 vendors at bottom of Teis (reset frequently); Blacksmith Forge Equipment (~330g per blue weapon) |
| Mythic Uniques | Undercity + Mythic Tribute of Armament + Portal Pranksters (4 in one run is real); Nemesis Portals from lair bosses; Butcher (post-3.0.x buff = best per-key rate) |
| GA Unique Charms | Only craftable via cube, never drop GA from the ground |
| Royal / Horadric Jewels | Sears Reach dungeon (top-right of Teis); reset and rush boss |
| Mythic Sparks | Tree of Whispers war plan top node (Resplendent Favor) on legendary whisper caches |
| Lair / Greater Lair Keys | War plans (Lair Boss tree); Hell Tide Orchard Gifts; World Bosses; Whisper Caches; Pale Marks → Boss Material Cache |
