## How this site is organized

**Quick Start** is the cheatsheet, the top-of-priority list you actually work through.

**Builds** is your personal tracker. Two classes (Barbarian, Warlock), two builds each, ranked within class. Same template on every build page so you can compare side by side.

**Reference (A–O)** is the deep library on universal mechanics: damage math, Horadric Cube, war plans, farming. Each letter is its own page so search returns precise hits instead of a 70KB scroll. Build pages reference the library rather than repeat it.

**Color callouts** appear throughout:

:::tip[Do]
Green callouts are recommended actions or proven patterns.
:::

:::danger[Don't]
Red callouts are confirmed mistakes that brick items or waste materials.
:::

:::caution[Caveat]
Amber callouts are watch-outs, qualifications, or edge cases worth knowing before you commit.
:::
