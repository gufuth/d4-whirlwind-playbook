title: 'L. Settings & QoL'
---

One-time setup, pays dividends every session.

### L1 — Core settings

| Setting | Where | Value |
|---|---|---|
| Pathfinder dotted line | Gameplay → Navigation Assist | On — the killer feature |
| HUD Compass (fallback) | Same menu | High-contrast green arrow |
| High Contrast Cursor | Graphics | On, Large, Green |
| **Advanced Tooltip Information** | Gameplay | **On (mandatory)** |
| Compare | Gameplay | On |
| Screen Shake / Combat Hit Flash | Graphics | Off |
| Reduced Strobing | Graphics | On |

### L2 — Controls (separate move from skill)

Default left-click combines move + interact + basic skill. You stop moving to cast basic skill when you don't want to. Fix:

- Controls → Combine Move/Interact/Skill = off
- Basic Skill slot → bind to 5 (or another free key)
- Move + Interact → one button (don't separate; you'll forget your interact key)

### L3 — Player highlight

Gameplay → Highlights → Display Player Highlight On. Pick a color that contrasts your outfit. Also set NPC, enemy, and object highlight colors.

### L4 — Loot filter (4 rules)

Activate from T3–T4 onwards.

| # | Action | Why |
|---|---|---|
| 1 | Hide All — Common + Magic + Rare | Visual cleanup. You're never picking up an 850 yellow at endgame. |
| 2 | Recolor — Codex Upgrade Check → **Purple** | Signals any item that would upgrade an aspect in your codex. |
| 3 | Recolor — Magic/Rare with ≥1 GA → **Red** | Top-tier crafting bases. |
| 4 | Show — Talisman Sets (Rare) — Charms + Horadric Seals | Rare set charms = 10 Infused Horadric Resin each. Stockpile. |

Bind a Loot Filter Toggle key (default L) so you can verify and quick-disable for treasure checks.

### L5 — Loot sounds

- In-Game Loot Sounds → minimum quality = Legendary (no sounds for 850 yellow drops)
- Play Ambient Loot → Uniques (audio cue for mythics/uniques)

### L6 — The `equipped` codex trick

:::tip[Do]
Codex of Power → search box → type

equipped

. Lists every aspect currently slotted on your gear. Quick max-roll audit without checking each piece.
:::

### L7 — Combat text

Set to *Crit Damage* for crit-scaling builds. Set to *Normal Damage* for DoT/petrify/non-crit builds (e.g., Druid Boulder rune) so you can see your actual highest damage instances.

### L8 — Waypoint favoriting

Right-click any waypoint on the map → favorite. Teleport key now sends you there by default. Saves 4–5 seconds per teleport, hundreds of times per session.
