:::tip[Big win]
**Nightmare Escalations + Astaroth at Torment 10** is the current best mythic seal source. Backup: Infernal Orbs + Bartuc (seed-hits loot pool). Skip Mephisto-spam. T10 not T12 — drop rates are identical, T10 clears 2x faster.
:::

The canonical 5-step circular farm. Self-sustains once running.

| # | Action | Goal |
|---|---|---|
| 1 | Stock Escalation NMD sigils (amalgamate 5 normal → 1 escalation) | Start with 3–4; loop self-sustains afterward |
| 2 | Run Escalation. At Asteroth's final tier, pick **Riches of Keys** / **Lair Keys** / **Greater Lair Keys** | First sustains the loop. Others stock your boss-key inventory. |
| 3 | Side-stream: Pale Marks → Bartering vendor → Boss Material Cache | Free extra lair keys |
| 4 | Burn lesser lair keys at any lesser boss (Varan is common) | Greater lair keys drop. Bial surprise-attack chance → Betrayer's Husks. Nemesis Portal chance after chest. |
| 5 | Spend greater lair keys at **Butcher** | Post-3.0.x Butcher has best mythic rate per key. Both chests can drop mythics back-to-back. |
| + | Spend Betrayer's Husks at Bial's Lair (2 husks → boosted loot from selected boss) | Bial has decent mythic seal rate |

:::tip[Do]
protocol

Nemesis Portal: 30-second window after the boss kill. Enter fast. Mythics drop. Can chain into a 2nd nemesis layer if lucky.
:::

## Sayur's recommended path (2026)

Per [Sayur's testing](https://www.youtube.com/watch?v=F7xU4cMoiAs), the current best mythic seal sources rank:

| # | Source | Why |
|---|---|---|
| 1 | **Nightmare Escalations → Astaroth (T10)** | High mythic + seal rate from Astaroth specifically, massive loot density (gold, mats, obols, uniques, ancestrals). Convert 5 regular NMDs to create one Escalation. Infinite supply. |
| 2 | **Infernal Orbs → Bartuc** | Bartuc has a seed-hits loot pool with strong mythic chance. |
| 3 | Boss farming (Mephisto etc.) | Works but loot density is much worse than the two above. Don't make it your only method. |

**T10 not T12.** Drop rates between T10 and T12 are practically identical. T10 clears 2x faster, more runs per hour = more drops.
