## Page template

Every build page carries the same eight sections so they're easy to compare side by side:

| Section | What goes here |
|---|---|
| Overview | One paragraph naming the source author and what makes this build different from siblings |
| Damage profile | Where the damage actually comes from. Skills, multipliers, scaling vectors |
| Key gear | Uniques + aspects priority order. Mandatory vs nice-to-have |
| Skills & rotation | The button-press order and what each skill does |
| Paragon, glyphs, talisman | Endgame nodes and the mandatory glyph levels |
| What to avoid | Common build-killers, anti-patterns, the things that brick the build |
| My notes | Where Ian tracks personal status, what's working, what isn't |
| Sources | Original guide + auxiliary research links |

## How to use this section

Open the class index page first. The class page ranks the builds and tells you which one fits which kind of play (push, speed farm, boss, mythic). Click into the specific build for the full breakdown.

The reference sections (A through O) still hold the universal mechanics: Horadric Cube, war plans, damage law, farming tricks. Build pages reference those sections rather than repeat them.
